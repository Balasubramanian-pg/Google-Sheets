# Google Sheets Basics Roadmap

## Introduction
Google Sheets is a powerful cloud-based spreadsheet application that allows you to create, edit, and share spreadsheets online. It's part of Google Workspace and offers a range of features for data management and analysis.

## Getting Started

### Creating a New Sheet
1. **Access Google Sheets**: Go to [Google Sheets](https://sheets.google.com) and sign in with your Google account.
2. **Create a New Sheet**: Click on the "+" button or go to File > New > Spreadsheet.

### Navigating the Interface
- **Menu Bar**: Familiarize yourself with the menu options: File, Edit, View, Insert, Format, Data, Tools, Add-ons, Help.
- **Toolbar**: Learn the icons for common actions like undo, redo, print, etc.
- **Formula Bar**: Where you can enter and edit formulas.
- **Sheet Tabs**: How to manage multiple sheets within a file.
- **Cell References**: Understanding rows, columns, and cell addresses.

## Data Entry and Formatting

### Entering Data
1. **Select a Cell**: Click on a cell to select it.
2. **Enter Data**: Type your data into the selected cell.
3. **Edit Data**: Click on a cell and press F2 or double-click to edit the data.

### Formatting Cells
1. **Font and Color**: Change font type, size, and color using the toolbar.
2. **Borders and Alignment**: Use the toolbar to add borders and align text.
3. **Number Format**: Change the format of numbers (currency, percentage, etc.).

### Basic Operations
1. **Copy and Paste**: Use Ctrl+C to copy and Ctrl+V to paste.
2. **Cut**: Use Ctrl+X to cut data.
3. **Basic Cell Formatting**: Use the toolbar to format cells.

## Basic Formulas and Functions

### Arithmetic Operations
1. **Addition**: `=A1 + B1`
2. **Subtraction**: `=A1 - B1`
3. **Multiplication**: `=A1 * B1`
4. **Division**: `=A1 / B1`

### Common Functions
1. **SUM**: `=SUM(A1:A10)` - Sums the values in cells A1 to A10.
2. **AVERAGE**: `=AVERAGE(A1:A10)` - Calculates the average of the values in cells A1 to A10.
3. **COUNT**: `=COUNT(A1:A10)` - Counts the number of numeric values in cells A1 to A10.

### Cell References
1. **Relative References**: References that change when copied (e.g., A1, B2).
2. **Absolute References**: References that stay constant when copied (e.g., $A$1, $B$2).

## Saving and Sharing

### Saving Your Work
1. **Auto-Save**: Google Sheets automatically saves your work.
2. **Version History**: Access previous versions via File > Version history.

### Sharing and Collaboration
1. **Share Button**: Click the "Share" button in the top-right corner.
2. **Collaborate**: Add email addresses to share the sheet and set permissions (view, edit, comment).
