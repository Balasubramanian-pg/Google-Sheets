## Sharing Basics

### How to Share a Sheet
1. Click the **Share** button in the top-right corner.
2. Enter email addresses or generate a shareable link.
3. Choose the appropriate permission level (View, Comment, Edit).

### Shareable Links
- Generate a link to share via email, chat, or social media.
- Adjust link permissions to **Restricted**, **Anyone with the link**, or **Public**.

## Permission Levels

### Viewer
- Can only view the sheet.
- Cannot edit or add comments.

### Commenter
- Can view and add comments.
- Cannot edit the sheet.

### Editor
- Can view, edit, and add comments.
- Can share the sheet with others.

## Real-Time Collaboration

### Live Editing
- Multiple users can edit the sheet simultaneously.
- Changes are saved automatically and visible in real-time.

### Presence Indicators
- See who else is viewing or editing the sheet with colored cursors and profile pictures.

### Chat and Communication
- Use built-in chat or comments to discuss changes without leaving the sheet.

## Comments and Action Items

### Adding Comments
1. Select a cell or range.
2. Right-click and choose **Comment**, or use **Insert > Comment**.
3. Tag collaborators using `@` to notify them.

### Action Items
- Assign tasks directly in comments (e.g., "@John, please review this data").
- Mark comments as resolved once tasks are completed.

### Notifications for Comments
- Get email notifications for new comments or replies.

## Version History

### Accessing Version History
- Use **File > Version history > See version history** to view and restore previous versions.

### Naming Versions
- Create named versions for important milestones (e.g., "Final Budget Approved").

### Restoring Old Versions
- Restore a previous version if changes need to be reverted.

## Notifications and Alerts

### Email Notifications
- Receive emails for edits, comments, or permission changes.
- Customize notification settings via **Tools > Notification rules**.

### Apps Script Alerts
- Use Google Apps Script to set up custom alerts (e.g., email notifications for specific cell changes).

## Sharing with Non-Google Users

### Sharing with Anyone
- Use the **Anyone with the link** option to share with users without Google accounts.
- Non-Google users may need to sign in with a temporary account or view in "preview" mode.

### Exporting for Non-Collaborators
- Export the sheet as **PDF, Excel, or CSV** for users who don’t need to edit.

## Advanced Sharing Techniques

### Sharing Specific Sheets or Ranges
- Protect sensitive data by sharing only specific sheets or ranges within a file.

### Using Filter Views
- Create and share **filter views** to allow collaborators to focus on relevant data without altering the original sheet.

### Embedding Sheets
- Embed sheets in websites or blogs using **File > Publish to the web**.

## Protecting and Locking Content

### Protecting Sheets and Ranges
- Use **Data > Protected sheets and ranges** to restrict editing for specific cells or sheets.
- Set permissions to allow only certain users to edit protected ranges.

### Locking Cells
- Use data validation or scripts to "lock" cells and prevent unintended edits.

## Best Practices for Collaboration

### Clear Communication
- Use comments and action items to clarify tasks and changes.
- Document major changes in a dedicated "Changelog" sheet.

### Organizing Data
- Use named ranges, tables, and consistent formatting to keep the sheet organized.
- Avoid merging cells, which can complicate collaboration.

### Regular Backups
- Periodically download backups of important sheets via **File > Download**.

### Training Collaborators
- Provide a quick guide or FAQ for new collaborators to ensure everyone understands how to use the sheet effectively.

## Next Steps
- Explore **Google Workspace add-ons** for enhanced collaboration features.
- Use **Google Apps Script** to automate repetitive collaboration tasks (e.g., sending reminders, updating shared sheets).

This roadmap covers everything you need to collaborate effectively in Google Sheets, from basic sharing to advanced techniques for managing large teams and complex workflows.
